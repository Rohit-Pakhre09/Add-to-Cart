<h1>🛒 Modern Shopping Cart Website</h1>

<p>
  This is a simple and dynamic shopping cart application built using HTML, CSS, and JavaScript. Users can browse products, 
  add them to their cart, and update item quantities. The cart data is saved using <code>localStorage</code> to ensure persistence 
  across page reloads.
</p>

<img src="https://github.com/Rohit-Pakhre09/Add-to-Cart/blob/5294e6159c4bea6ff73f33d517a70e954e5e5e75/Add%20to%20Cart/Assets/Add%20to%20Cart.png" alt="Project">

<h2>🚀 Features</h2>
<ul>
  <li>🛍️ Display a list of products fetched from a local <code>products.json</code> file.</li>
  <li>🛒 Users can add products to the cart.</li>
  <li>➕➖ Increase or decrease the quantity of products in the cart.</li>
  <li>💾 Persistent cart data using <code>localStorage</code>.</li>
  <li>📱 Fully responsive design for mobile, tablet, and desktop views.</li>
  <li>🎨 Clean and intuitive user interface with modern design elements.</li>
</ul>

<h2>🛠️ Tech Stack</h2>
<ul>
  <li>HTML5, CSS3, and JavaScript (ES6+)</li>
  <li><code>localStorage</code> for persisting cart data</li>
  <li>Responsive design with media queries and flexbox/grid</li>
</ul>

<h2>📂 Project Structure</h2>
<ul>
  <li><code>index.html</code> - Main HTML file for the project</li>
  <li><code>style.css</code> - CSS file for layout and design</li>
  <li><code>script.js</code> - JavaScript file handling dynamic behavior and cart functionality</li>
  <li><code>products.json</code> - JSON file containing the list of products</li>
</ul>
